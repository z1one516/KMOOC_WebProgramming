This div is for testing. 웹용 아이콘 글꼴 머티리얼 아이콘 글꼴은 머티리얼 아이콘을 웹 프로젝트에 통합하는 가장 쉬운 방법입니다. Google에서는 모든 머티리얼 아이콘을 최신 글꼴의 글꼴 렌더링 기능을 활용하는 단일 글꼴로 패키징하여 웹 개발자가 코드 몇 줄만으로 이러한 아이콘을 쉽게 통합할 수 있도록 했습니다. 글꼴 사용은 가장 편리한 방법일 뿐만 아니라 효율적이고 멋집니다. 아이콘 900개 이상을 작은 파일 하나에서 모두 제공합니다. Google 웹 글꼴 서버에서 게재되거나 자체 호스팅될 수 있습니다. 모든 최신 웹브라우저에서 지원됩니다. 색상, 크기, 위치가 모두 CSS로 지정됩니다. 벡터 기반: 모든 크기, 레티나 디스플레이, 저dpi 디스플레이 화면에서 멋지게 표시됩니다. 아이콘 글꼴의 두께는 가장 작은 woff2 형식에서 42KB, 표준 woff 형식에서 56KB에 불과합니다. 이에 비해 gzip으로 압축된 SVG 파일은 일반적으로 크기가 약 62KB이지만, 필요한 아이콘만 기호 스프라이트가 포함된 단일 SVG 파일로 컴파일하여 크기를 크게 줄일 수 있습니다.
